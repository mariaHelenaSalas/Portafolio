## 📥 Instalación y ejecución
1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/mariaHelenaSalas/Portafolio.git
   ```
2. **Abrir la carpeta del proyecto** en tu editor de código favorito.  
3. **Ejecutar `index.html`** en un navegador web.
